Welcome

Welcome to my personal website. My name is Bas van Schoonhoven. I live in the Netherlands and work as a senior software developer at OCLC. Previously I have done technical and policy research at TNO ICT, and worked as a software developer at DE REE Archiefsystemen.

Professionally I mostly do Java and MySQL backend development with some front-end work mixed in. As a hobby I create prototypes and toys using Rust, Java and OpenGL and sometimes other languages and frameworks.

This site acts as a portfolio and as a place for sharing things that interest me in my Blog.
